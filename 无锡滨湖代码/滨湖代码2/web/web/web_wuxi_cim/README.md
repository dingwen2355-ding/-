
# ai_city

## node.js 版本
v14.17.1

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### 启动方式
```
1、配置ipConfig.js中的用户名、密码；
2、安装依赖并启动项目；
```
